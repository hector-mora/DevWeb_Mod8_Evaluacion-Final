Con las siguientes instucciones se ejecutara el servidor "http://localhost:3000", con una estructura MERN (Mongo-Express-ReactJS-Node) donde el archivo index.js tenemos la configuracion del servidor con express y en la carpeta server los modelos de mongoose y las rutas; por otro lado en la carpeta (static) "public" tenemos el frontend de ReactJS con el ultimo build realizado en la carpeta "public" que express buscara como referencia de front end.

Simplemente se debe:

=> Crear la BD en mongodb (carpeta BASE DE DATOS MONGO)

=> Abrir la consola de comandos	
	-ejecutar mongod

=> Abrir la consola de comandos
	-ubicarse en la carpeta de "cd ...TiendaReactJs\EvaluacionFinal"
	-ejecutar node index.js

=> Abrir Google Chrome en la direccion "http://localhost:3000"
